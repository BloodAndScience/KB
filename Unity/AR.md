# ARKit Face Tracking

## Features

1. Face Anchor
2. Face Mesh
3. Face expression coefficient
4. Face BlendShapes
    - SkinRenderer 51 BlendShapes  vertices

public delegate void ARFaceAnchorUpdated(ARFaceAnchor anchorData);
